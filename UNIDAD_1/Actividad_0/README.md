# Actividad_1

Este proyecto grafica una señal seno (continua y discreta).

## Archivos:

- `main.py`: corre el programa principal.
- `src/signal.py`: genera las señales.
- `src/graphs.py`: muestra las gráficas.

# Cómo usarlo:

1. Instala las dependencias:
pip install -r requirements.txt


2. Corre el programa:
python main.py


## Requisitos:
- Python 3
- numpy
- matplotlib
