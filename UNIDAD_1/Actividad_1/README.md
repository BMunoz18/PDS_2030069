# Actividad_1

Este proyecto grafica 4 funciones (Senoidal, Exponencial por escalon, Triangular y Cuadrada) de manera continua, discreta y sobrepuestas, una a una.

## Archivos necesarios:

- `main.py`: corre el programa principal.
- `src/signals.py`: genera las señales.
- `src/graphs.py`: muestra las gráficas.

# Cómo usarlo:

1. Instala las dependencias:
pip install -r requirements.txt


2. Corre el programa:
python main.py


## Requisitos:
- Python 3
- numpy
- matplotlib
- scipy
